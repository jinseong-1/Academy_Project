# CSS


## css Inline 방식
<b>color:~~;</b> : 글자 색 지정<br/>
<b>선택자</b> : 상단에서 어떤 태그요소에 적용시킬지 쓰는 것<br/>
<b>{}</b> : 범위지정

### 내부 스타일 방식
___
example)<br/>
    style<br/>
        h1{<br/>
            background-color:
            ~~~~~;<br/>
        }<br/>
    /style<br/>
    
    tage가 h1인 모든 곳에 ~~~~~의 color가 적용
스타일의 주석 : /* */<br/>
색상표현 : RGB는 10진법으로 0~255, 
16진법으로 0~ff까지

### 외부 스타일 방식
___
rel : 외부문서와 연결해주는 링크<br/>
ex) link rel="stylesheet" href="~~~~" : 참고하고자하는 ~~~파일<br/>
스타일시트 우선순위 : 파일이 나눠져있다고 우선권이 있는게 아니라 위에서부터 덮어써서 내려오는 것<br/>
여러 페이지에서 공통적으로 사용될때 사용되는 것.<br/>
css 선택자<br/>
(*) : 사칙연산을 제외하면 전체를 의미<br/>
<b>스타일을 적용할 선택자를 찾아가는 기본적인 방식</b><br/>
아이디 선택자 : #아이디
 _-> 아이디는 중복이 있어선 안되지만 문법에선 이상이 없어 구현은 됨<br/>
클래스 선택자 : .클래스<br/>
$ 는 1에서부터 점차 숫자가 올라가게 생성가능<br/>
padding : 컨텐츠간의 간격을 의미<br/>
margig : 가장 바깥의 간격 → 이게 0이면 딱붙게 됨<br/>
margin과 padding을 이용해서 간격을 조절할수있음<br/>

## id와 클래스를 이용한 스타일 적용
style<br/>
        *{
            color:greenyellow;
        }

        h1{
            color:red;
        }

        #bg{
            background-color: skyblue;
        }

        #color{
            color:blue;
        }

        .size{
            /* 글자 크기 px로 지정 */
            font-size:50px;
        }
        .color{
            color:pink;
        }
</style>
- 복잡하게 되어있지만
#은 bg, color인 아이디에 각각 적용
.은 size, color라고 적혀있는 클래스에 각각 적용
- 클래스에서 공백을 기준으로 다른 클래스를 같이 적용
- class="size color" → size와 color를 동시에 적용
But id 속성엔 위 클래스와 같이 적용이 되지않고 하나만 적용됨
- 선택자에 따른 style 적용 우선순위
inline > id 선택자 > class 선택자 > type 선택자
좀 더 디테일한 것을 적용함
